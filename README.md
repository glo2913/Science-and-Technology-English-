## 🌟 核心特色

- **🎯 多維度複習模式**：包含 `依序`、`隨機`、`20題測驗` 及 `收藏模式`。
- **🔊 沉浸式體驗**：支援 **TTS 發音** 與 **三種視覺主題**（極簡、深色、遊戲）。
- **⌨️ 快捷操作**：全鍵盤支援 `Space`、`方向鍵`、`R/S` 鍵，效率加倍。
- **💾 數據同步**：使用 `LocalStorage` 紀錄收藏，`PapaParse` 動態載入 CSV。
